# Habit Logger

This is .NET 7 C# console application.

Users can use it to log their daily water-drinking habits.


# Given Requirements

This is an application where user will register water drinking habit.

- [x]  This habit will be tracked by quantity (ex. number of water glasses a day)
- [x]  The application should store and retrieve data from a real database
- [x]  When the application starts, it should create a **sqlite database**, if one isn¡¯t present.
- [x]  It should also create a table in the database, where the habit will be logged.
- [x]  The app should show the user a menu of options.
- [x]  The users should be able to insert, delete, update and view their logged habit.
- [x]  You should handle all possible errors so that the application never crashes.
- [x]  The application should only be terminated when the user inserts 0.
- [x]  Interact with the database using raw SQL. Dose not use mappers such as Entity Framework.
- [x]  Contain a Read Me file explains how app works.

# Features

* User Main Menu
* Insert, Delete, Select, Update water habit record
